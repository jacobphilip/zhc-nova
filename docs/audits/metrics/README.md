# Metrics Reports

Closed-loop metrics outputs generated by `scripts/metrics_report.py`.

## Files

- `latest_metrics.json`: rolling machine-readable summary.
- `latest_metrics.md`: rolling human-readable summary.
- `<date>-<iteration>_metrics.{json,md}`: archived snapshots.

## Commands

```bash
make metrics

python3 scripts/metrics_report.py \
  --days 7 \
  --iteration 2026-02-25-v1.1 \
  --output-json docs/audits/metrics/2026-02-25-v1_1_metrics.json \
  --output-md docs/audits/metrics/2026-02-25-v1_1_metrics.md
```
